
# 直线画法


### 炮灰
- 斜截式
	- y = kx + b 
	- 这种写法，人来读的话可能还好，但是从代码实现的角度来说，还要先把 k 和 b 算出来
	- 滚蛋
- 两点式
	- x = k (x2-x1) + x1
	- y = k (y2-y2) + y1
	- 将 k 从 0 -> 1 按照 0.01 （或其它step）进行步进即可
	- 能用，但是比较low

### DDA
- 确定step
	- 从 dx 和 dy 中找最大值 step （取最大值是为了避免跳格， 允许对一个像素重复设置，但是不能漏）
	- 然后 dx 和 dy 分别处以 step，作为两个方向的步进长度
- 循环中，分别加上各自的步进长度即可
- 循环过程中有浮点计算，比较废
